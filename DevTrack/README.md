# DevTrack

DevTrack - A personal developer productivity dashboard to manage tasks, track time, and hit daily goals.

## Features
- Task Manager with localStorage persistence
- Focus Timer with session tracking
- Live Stats Panel
- Dark / Light theme toggle
- Export tasks to CSV

## How to Run
Open index.html in any modern browser. No build step needed.
